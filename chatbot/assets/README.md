# Video Avatar Placeholder

Este archivo es un marcador de posición para el video del chatbot.

## Instrucciones

Agrega tu video del chatbot aquí con el nombre:
**chatbot-avatar.webm**

Para instrucciones detalladas, consulta:
`VIDEO_INSTRUCTIONS.md` en la carpeta principal del chatbot.

## Formato Requerido

- **Nombre**: chatbot-avatar.webm
- **Formato**: WebM con VP9
- **Transparencia**: Canal alpha o fondo verde (se eliminará automáticamente)
- **Dimensiones**: 300x400px recomendado
